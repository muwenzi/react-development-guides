# 命名规范

## 文件夹命名

{% hint style="success" %}
**推荐**：React 组件的文件夹命名都是 **大驼峰 + 单数**，其他文件夹都是 **小驼峰 + 复数**。
{% endhint %}

示例：

```text
Header
Button
components
widgets
utils
states
apis
libs
```

## 文件命名

{% hint style="success" %}
**推荐**：文件命名都是 **小驼峰**，单复数看情形。
{% endhint %}

## 方法/变量命名

{% hint style="success" %}
**推荐**：方法名都是 **小驼峰**，单复数看情形。
{% endhint %}

{% hint style="success" %}
**推荐**：变量名除常量外都是 **小驼峰**，单复数看情形。
{% endhint %}

{% hint style="success" %}
**推荐**：常量名都是 **全大写字母 + 下划线分隔**，单复数看情形。
{% endhint %}

示例：

```text
UNIFIED_STATES
LINES_SET_FILTER
```
